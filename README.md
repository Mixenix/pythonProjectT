# Flappy Bird на Pygame

Клон классической игры Flappy Bird на Python с системой рекордов.

## Особенности

- Физика прыжка и столкновений
- Генерация случайных препятствий
- Сохранение лучшего результата в файл `data/score.txt`
- Анимированные спрайты и звуки

## Стек

- **Python 3.7+**
- **Pygame 2**

## Установка и запуск

```bash
pip install pygame
python main.py
```

Либо запустить готовый `main.exe` (Windows), поместив его в одну папку с каталогом `data`.
